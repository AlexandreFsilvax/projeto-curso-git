Qual comando você utiliza para retirar um arquivo adicionado para commitar após utilizar o git add? git reset HEAD NOME_DO_ARQUIVO
Quais dados são commitados quando o comando 'git log' é usado? O id, um branch, um data, o e-mail e o nome do autor.
QUal comando se deve usar quado você quer resetar como mudanças de um arquivo? git checkout --NOME_DO_ARQUIVO
Sequencia para unir mudanças remotas e enviar para o repositório no branch master? Selecionar Brach, git merge NOME_DA_BRANCH, git push origin master
Comando para enviar nossa filial do repositório local para o repositório remoto? git push origin NOME_DA_BRANCH
Quais ramos são listados no repositório do GitHub? Todas que foram feitas push
Qual o comando usado para editar a mensagem do último commit? git commit --amend
Qual comando se utiliza para ver como transformadas no arquivo? git diff NOME_DO_ARQUIVO

O que é Deploy? É colocar o projeto em produção, colocar no ar. Quando sobe os arquivos para um servidor. Pode ser manual: pequeno ajuste no algoritmo, parcialmente automatizado: quando faz um push e atualiza o servidor de hospedagem do git. Automatizado: Uma forma de integração contínua como Jenkins para realizar teste automaticamente. Faz mais sentido em Desenvolvimento Web, quando queremos rodar o projeto já finalizado num ambiente acessível para todos. Heroku é uma empresa que vê projetos no git hub e testa-os.
