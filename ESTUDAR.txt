Quais dados do commit são listados quando o comando 'git log' é usado? O id, a branch, a data, o e-mail e o nome do autor.
QUal comando se deve utilizar quado você quer resetar as alterações de um arquivo? git checkout --NOME_DO_ARQUIVO
Sequencia para unir alterações commitadas e enviar para o repositório remoto na branch master? Selecionar Brach, git merge NOME_DA_BRANCH, git push origin master
Comando para enviar a nossa branch do repositório local para o repositório remoto? git push origin NOME_DA_BRANCH
Quais branches são listadas no repositório do GitHub? Todas que foram feitas push
Qual o comando utilizado para editar a mensagem do último commit? git commit --amend
Qual comando se utiliza para ver as modificações feitas no arquivo? git diff NOME_DO_ARQUIVO

O que é Deploy? É colocar o projeto em produção, colocar no ar. Quando sobe os arquivos para um servidor. Pode ser manual : pequeno ajuste no algoritmo, parcialmente automatizado: quando faz um push e atualiza o servidor de hospedagem do git . Automatizado: Uma forma de integgração contínua como Jenkins para realizar teste automaticamente. Faz mais sentido em Desenvolvimento Web, quando queremos rodar o projeto já finalizado num ambiente acessível para todos. Heroku é uma empresa que vê projetos no git hub e testa-os.
